## Overview

In this series of blog posts we will explore the common sorting algorithms, the theory behind them, as well as how to implement them using python. The inspiration and code base can be largely attributed to this wonderful collection on github, [The Algorithms - Python](https://github.com/TheAlgorithms/Python), while the theoritical side is built upon many sources, and will be mentioned as we go along.


The following algorithms will be covered in this series:

- Bubble Sort
- Selection Sort
- Insertion Sort
- Merge Sort
- Quick Sort
- Heap Sort
- Tim Sort
